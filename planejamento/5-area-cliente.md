# Fase 5: Área do Cliente

O hub central onde o cliente acompanha sua "fábrica particular".

## 👤 Detalhamento de Funcionalidades

### 1. Dashboard de Pedidos
- **Cards de Status:** Cada pedido exibe um badge colorido:
    - `Blue`: Pago / Em Análise
    - `Orange`: Imprimindo (Gera expectativa!)
    - `Green`: Pronto para Envio
- **Timeline Detalhada:** Ao clicar em um pedido, mostra o histórico completo com datas e mensagens do operador.

### 2. Gestão de Arquivos
- Galeria de arquivos STL já enviados anteriormente pelo usuário.
- Botão "Imprimir Novamente" que pré-configura um novo orçamento/pedido com o mesmo arquivo.

### 3. Programa de Fidelidade
- Exibição do "Saldo de Pontos".
- Regra: R$ 10,00 gastos = 1 ponto.
- Visualização de quanto falta para o próximo cupom de desconto.

## 📌 Meta de Engajamento
- Criar um ambiente onde o cliente se sinta parte de uma comunidade tecnológica, não apenas um comprador de site comum.
