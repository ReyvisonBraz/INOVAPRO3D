# Fase 6: Painel Admin — Essentials

O coração da operação produtiva.

## 🛠️ Ferramentas de Gestão

### 1. O Kanban de Produção
- Colunas: `Pendente`, `Em Análise`, `Na Fila (Impressão)`, `Imprimindo`, `Pós-Processamento`, `Pronto`, `Enviado`.
- Ação: Ao mover para "Imprimindo", o operador seleciona qual máquina está realizando o trabalho.

### 2. Central de Orçamentos (Análise de STL)
- O admin abre o pedido customizado.
- O visualizador 3D mostra as dimensões do arquivo.
- O admin insere: "Valor da Peça", "Dificuldade", "Estimativa de Gramas".
- Ao clicar em "Enviar Proposta", o sistema envia o e-mail/zap e muda o status para o cliente aprovar.

### 3. Upload de Prova Fotográfica
- O admin tira uma foto da peça pronta e faz upload direto no pedido.
- Essa foto aparece instantaneamente na área do cliente.

## 📌 Meta de Operação
- Reduzir o tempo de resposta do admin e organizar o chão de fábrica digitalmente. Precisão máxima no cálculo de custo.
